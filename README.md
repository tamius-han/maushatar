# maushatar
Discdord transcriber bot, based on Mozilla's deep speech.

# Installing

## Requirements:

* Node 14
* npm

## Pre-installation

Acquire `env.ts` and put it in `/app/env/env.ts`. See `README-ENV.md` for required and supported parameters. 

## Compiling

```sh
npm ci
npm run build
```
